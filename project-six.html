Project 6: Cisco Network Design Labs

Enterprise network design and configuration exercises completed using Cisco Packet Tracer, covering VLANs, routing, and IPv4/IPv6 addressing.

Project Description

This collection of Cisco Packet Tracer labs covers enterprise network design, configuration, and troubleshooting using Cisco IOS commands. The labs progress from foundational LAN switching concepts through advanced WAN routing configurations, building practical networking skills directly applicable to real-world IT and cybersecurity environments.

Key labs include multi-switch VLAN segmentation, inter-VLAN routing using router sub-interfaces, trunk interface configuration, static routing over WAN serial connections, IPv4 subnetting with CIDR notation, and IPv6 addressing and routing exercises.

Technologies & Skills Used

  • Cisco Packet Tracer for network simulation and configuration
  • Cisco IOS CLI commands for switch and router configuration
  • VLAN creation, assignment, and segmentation across multiple switches
  • Trunk interfaces for inter-switch VLAN communication
  • Router sub-interfaces for inter-VLAN routing (Router-on-a-Stick)
  • Static routing configuration over WAN serial connections
  • IPv4 subnetting, CIDR notation, and subnet planning
  • IPv6 addressing, prefix notation, and static routing

Relevance to Cybersecurity & IT

  • VLAN segmentation is a core network security control used to isolate sensitive systems and limit lateral movement in enterprise environments
  • Understanding routing protocols and WAN connectivity is foundational for network security analysis and firewall rule design
  • IPv6 fluency is increasingly required as enterprise networks transition away from IPv4
  • Hands-on CLI experience prepares for real-world switch and router administration in enterprise IT roles
