Project 5: CineTrack — Full-Stack Movie & TV Tracker

A full-stack MERN web application for tracking movies and TV shows, built as a team project for CIS 4004 (Web-Based IT) at UCF.

Project Description

CineTrack is a full-stack movie and TV show tracking application built with the MERN stack (MongoDB, Express, React, Node.js) by Group 5 in CIS 4004 Spring 2026. The application allows users to search for movies and TV shows, add them to a personalized Watchlist, mark items as watched, and manage their viewing history. The platform supports two user roles with different permission levels and five core entities connected through many-to-many Watchlist relationships.

My role on the team was frontend lead. I was responsible for building the complete React UI, including all components, page layouts, routing, and user-facing interactions. I also managed project documentation, the planning phase, the README, and coordinated team collaboration through GitHub and Discord.

Technologies & Skills Used

  • React (functional components, hooks, React Router)
  • Node.js and Express for the backend REST API
  • MongoDB with Mongoose for data modeling
  • Two user roles with role-based UI rendering
  • Five entities with many-to-many Watchlist relationships and full CRUD operations
  • GitHub for version control and team collaboration
  • Discord for team communication and coordination
  • Project documentation: README, planning materials, and presentation
